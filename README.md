# WebAPP
Web application for job seekers.

# Featuers
* Register and login system
* Password reset
* Email verfication
* CV generator
* Post a job
* Search a job offer
* Apply for a job
* Displaying applied job offers
* Displaying posted job offers
* Edit/Delete posted job offers
